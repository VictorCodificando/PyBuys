HAY QUE TENER PYTHON INSTALADO
1- Abrir la cmd en src con permisos de adminstrador
2- Poner el siguiente comando para ejecutar el entorno virtual:
pybuys_env\Scripts\activate.bat

3-Luego simplemente llamamos al .bat de ejecutar y ya el solo se encarga de ejecutar todo lo necesario
EJECUTAR.bat
